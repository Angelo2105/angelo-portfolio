Task Automation

Overview

I developed this Python automation system with the help of AI to eliminate repetitive document management tasks in a specific software that the company I worked for used. The application combines browser automation with Playwright and real-time file monitoring to download, organize, rename, and process construction documents automatically.

Key Features

  • Automated document downloads using Playwright.
  • Monitored the Windows Downloads folder in real time.
  • Automatically extracted ZIP archives.
  • Renamed files into standardized naming conventions.
  • Moved files to their correct project directories.
  • Reduced repetitive manual work and improved consistency.
