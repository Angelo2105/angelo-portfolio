Task Automation

Overview

I developed this Python automation system to eliminate repetitive document management tasks in Klaros. The application combines browser automation with Playwright and real-time file monitoring to download, organize, rename, and process construction documents automatically.

Key Features

  • Automated document downloads using Playwright.
  • Monitored the Windows Downloads folder in real time.
  • Automatically extracted ZIP archives.
  • Renamed files into standardized naming conventions.
  • Moved files to their correct project directories.
  • Reduced repetitive manual work and improved consistency.
